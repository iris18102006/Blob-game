# BlobPet
A cute, interactive command-line virtual pet written in Java. Feed it, play with it, pet it, and watch its mood change in real time!
 

# Features
Mood-based ASCII faces that react to your choices
Persistent save file — your blob remembers you between sessions
Colorful terminal output for extra cuteness
No dependencies — just plain Java

# Prerequisites
Java 8 or higher

# How to play
1.Start the program and name your blob (or load your existing one)

2.Choose an action from the menu:

Feed — reduces hunger
Play — boosts happiness but costs energy
Sleep — restores energy
Pet — makes your blob happy
Rename — give your blob a new name
Exit — saves and quits

3.Watch the face change after every choice!
